■ インストール方法

【フォルダの場合】
Chromeの[三]から設定→拡張機能→デベロッパーモードにチェックを入れ、
ダウンロードしたフォルダを指定。
Library Searchのアイコンがでれば成功しています。

【crxファイルの場合】(crxは再起動等すると、Chromeのセキュリティ関係で無効化するようです。)
Chromeの[三]から設定→拡張機能→デベロッパーモードにチェックを入れ、
ダウンロードしたcrxファイルを指定
Library Searchのアイコンがでれば成功しています。


■ 使い方

右クリックから使うアプリです。

【初期設定】拡張機能のオプションで検索したい都道府県を選択しSAVEボタンを押して下さい。(SAVE後は設定画面を閉じて構いません)
デフォルトは東京です。東京の方は、設定する必要はありません。

1.検索したいワードをドラッグして下さい。
2.右クリックをおします。
(※ドラッグをしないとでてきません)

オプション画面は拡張機能の管理画面から行ってください。
ブラウザ機能のキャッシュ削除を行うと設定が消えます。その際は、再設定して下さい。

■ 概要

カーリルローカルを利用したアプリです。

通常、ブラウズ中に「この本、図書館にないかな？」と思っても、
1.図書館のサイトを開く
2.検索ページに探して移動
3.タイトルを入力する(忘れた場合は元のページに戻らなければいけない。)
4.検索..
5.見つからない場合、他の図書館を..
と、これだけのステップを踏まなければなりません。

このプラグインを使えば、
1.タイトルをドラッグする
2.右クリックで検索
と大幅に省略でき、尚且つ、気になる本を見つけたサイト上で全ての作業が済みます。
タイトルも間違える心配がありません。
また、県に登録されている全ての図書館を高速に検索しますので、ストレスフリーです。
