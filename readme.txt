■ インストール方法

【crxファイルの場合】
Chromeの[三]から設定→拡張機能→デベロッパーモードにチェックを入れ、
ダウンロードしたcrxファイルを指定
Library Searchのアイコンがでれば成功しています。

【フォルダの場合】
Chromeの[三]から設定→拡張機能→デベロッパーモードにチェックを入れ、
ダウンロードしたフォルダを指定。
Library Searchのアイコンがでれば成功しています。

■ 使い方

右クリックから使うアプリです。
【初期設定：】拡張機能のオプションで検索したい都道府県を選択しSAVEボタンを押して下さい。
デフォルトは東京です。東京の方は、設定する必要はありません。

1.検索したいワードをドラッグして下さい。
2.右クリックをおします。
(※ドラッグをしないとでてきません)

オプション画面は拡張機能の管理画面から行ってください。

■ 概要

カーリルローカルを利用したアプリです。

通常、ブラウズ中に「この本、図書館にないかな？」と思っても、
図書館のサイトを開く→検索ページに探して移動→タイトルを入力する→検索..見つからない場合、他の図書館を..]というステップを踏まなければなりません。

このプラグインを使えば、
「タイトルをドラッグする→右クリ検索]
と大幅に省略できます。

個人的に「プログミングの本買うと高いんだよなぁー・・・・図書館ないかな (ﾎﾞｿ..」と思って気軽に調べるために作成しました。
