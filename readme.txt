Library Search

***************

Google Extension
version 1.0.7
作成者： mit-nk

***************

インストール方法:

    フォルダの場合：
Chromeの[三]から設定→拡張機能→デベロッパーモードにチェックを入れ、
ダウンロードしたフォルダを指定。
Library Searchのアイコンがでれば成功しています。

    crxファイルの場合：
Chromeの[三]から設定→拡張機能→デベロッパーモードにチェックを入れ、
ダウンロードしたcrxファイルを指定
Library Searchのアイコンがでれば成功しています。
(現在、Chromeではcrxは無効になっている様です。)

使い方：

【初期設定：】拡張機能のオプションで検索したい都道府県を選択しSAVEボタンを押して下さい。
デフォルトは東京ですので、その場合は、なにも設定する必要はありません。

1.検索したいワードをドラッグして下さい。
2.右クリックをおします。

概要:

カーリルローカルを利用したChrome extension。

ブラウズ中、「この本、図書館にないかな？」と思っても、
[気になる本が見つかる→図書館のサイトを開く→検索ページに移動→タイトルを入力する→検索..見つからない場合、他の図書館を..]というステップを踏まなければなりません。

このプラグインを使えば、「（気になる本が見つかる）→タイトルをドラッグする→右クリ検索]
と大幅に省略できます。

個人的に「プログミングの本高いんだよなぁー・・・・図書館ないかな (ﾎﾞｿ..」と思って気軽に調べるために作成しました。
